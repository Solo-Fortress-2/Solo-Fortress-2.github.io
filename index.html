Welcome to Solo Fortress 2

Solo Fortress 2 is a work-in-progress free-for-all Team Fortress 2 Sourcemod by Kepler, based on the latest Team Fortress 2 SDK. It aims to improve some core Team Fortress 2 gameplay for a better deathmatch experience.

Our current features

Here are the current features of Solo Fortress 2:

  • Randomized spawn points

  • Disguisable props

  • Improved Huntsman hitbox detection

  • Buffed weapons and items
  • Ultralight web renderer (W.I.P.)
  • Web-based main menu (can be disabled dw)
